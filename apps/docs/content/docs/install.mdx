## Getting Started

To get started, follow the steps below.

<Steps>
<Step title="Install Package">

Install `audio-kit` with your package manager of choice.

<Tabs items={[
  { label: "npm", value: "npm install audio-kit" },
  { label: "yarn", value: "yarn add audio-kit" },
  { label: "pnpm", value: "pnpm add audio-kit" },
  { label: "bun", value: "bun add audio-kit" },
]} />

</Step>
<Step title="Resume AudioContext">

Browsers block audio until a user gesture fires. Call `ensureReady` once to resume the `AudioContext`. Place this in a click or pointer handler before playing any sound.

```ts title="setup.ts"
import { ensureReady } from "audio-kit";

button.addEventListener("click", async () => {
  await ensureReady();
});
```

After the first call resolves, subsequent calls return immediately.

</Step>
<Step title="Play Your First Sound">

Define a sound as a plain object and call the returned function to play it.

```ts title="app.ts"
import { defineSound } from "audio-kit";

const pop = defineSound({
  source: { type: "sine", frequency: { start: 400, end: 150 } },
  envelope: { decay: 0.05 },
  gain: 0.35,
});

pop();
```

`defineSound` compiles the definition once and returns a zero-allocation play function. Call it as many times as you need.

</Step>
</Steps>

## React

Install the same package. React hooks are exported from `audio-kit/react`.

<Steps>
<Step title="Add SoundProvider">

Wrap your app (or a subtree) with `SoundProvider` to manage global enabled/volume state. When the user's OS has `prefers-reduced-motion: reduce` enabled, `SoundProvider` automatically disables sound.

```tsx title="app.tsx"
import { SoundProvider } from "audio-kit/react";

function App() {
  const [enabled, setEnabled] = useState(true);
  const [volume, setVolume] = useState(0.8);

  return (
    <SoundProvider
      enabled={enabled}
      volume={volume}
      onEnabledChange={setEnabled}
      onVolumeChange={setVolume}
    >
      {children}
    </SoundProvider>
  );
}
```

</Step>
<Step title="Use Sounds">

`useSound` takes a sound definition and returns a stable play function. The play function returns a `VoiceHandle` you can use to stop the voice, or `undefined` when sound is disabled.

```tsx title="button.tsx"
import { useSound } from "audio-kit/react";

function SaveButton() {
  const play = useSound({
    source: { type: "sine", frequency: 880 },
    envelope: { decay: 0.1 },
    gain: 0.3,
  });

  return <button onClick={() => play()}>Save</button>;
}
```

</Step>
</Steps>

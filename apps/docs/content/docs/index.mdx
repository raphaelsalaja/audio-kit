---
title: Overview
description: Declarative audio synthesis for the web.
---

Audio Kit lets you describe sounds as plain objects and play them with a single function call. No imperative node wiring. No class hierarchies. Just JSON in, audio out.

```ts
import { defineSound } from "audio-kit";

const pop = defineSound({
  source: { type: "sine", frequency: { start: 400, end: 150 } },
  envelope: { decay: 0.05 },
  gain: 0.35,
});

pop();
```

A sound definition is a plain object describing a signal path. Pick a source, shape it with an envelope, run it through filters and effects, and play it. Layers let you stack multiple signals into one sound. Packs let you bundle sounds as JSON and load them at runtime.

Audio Kit handles the Web Audio plumbing: one shared `AudioContext`, automatic node cleanup, sample-accurate scheduling, and a `prefers-reduced-motion` opt-out. Works with vanilla JS, React, or any framework.

## Declarative sounds

Describe a sound as a plain object. `defineSound` compiles it once and returns a play function.

```ts
const play = defineSound({
  source: { type: "sine", frequency: 440 },
  envelope: { decay: 0.1 },
  gain: 0.3,
});

play();
```

## ADSR envelopes

Shape amplitude over time with attack, decay, sustain, and release. Only `decay` is required.

```ts
{
  envelope: { attack: 0.01, decay: 0.3, sustain: 0.4, release: 0.5 }
}
```

## Filters

Biquad filters (lowpass, highpass, bandpass, notch, allpass, lowshelf, highshelf, peaking) with optional filter envelope. IIR filters with custom coefficients.

```ts
{
  filter: { type: "lowpass", frequency: 800, Q: 2 }
}
```

## 14 built-in effects

Reverb, convolver, delay, distortion, chorus, flanger, phaser, tremolo, vibrato, bitcrusher, compressor, EQ, gain, and stereo pan. Effects chain in order, per-layer or global.

```ts
{
  effects: [
    { type: "reverb", decay: 2, mix: 0.3 },
    { type: "delay", time: 0.25, feedback: 0.4, mix: 0.2 },
  ]
}
```

## LFO modulation

Low-frequency oscillators modulate any target parameter: frequency, detune, gain, pan, filter cutoff, filter Q, filter gain, filter detune, or playback rate.

```ts
{
  lfo: { type: "sine", frequency: 5, depth: 50, target: "frequency" }
}
```

## Multi-layer sounds

Stack multiple signals with independent sources, envelopes, filters, effects, and gain. Each layer can have its own onset delay.

```ts
{
  layers: [
    { source: { type: "sine", frequency: 440 }, envelope: { decay: 0.3 }, gain: 0.3 },
    { source: { type: "noise", color: "white" }, envelope: { decay: 0.05 }, gain: 0.15, onset: 0 },
  ]
}
```

## Spatial audio

Position sounds in 3D space with `Panner3D`. Supports HRTF and equalpower panning models, distance attenuation, and directional cones.

```ts
{
  panner3d: { positionX: 2, positionY: 0, positionZ: -1, panningModel: "HRTF" }
}
```

## Sound packs

Bundle sounds into a pack and load them at runtime. Packs are JSON files with a manifest and named sound definitions.

```ts
import { definePack, loadPack } from "audio-kit";

const pack = definePack({
  name: "ui-sounds",
  sounds: {
    click: { source: { type: "sine", frequency: 800 }, envelope: { decay: 0.03 }, gain: 0.2 },
    hover: { source: { type: "sine", frequency: 600 }, envelope: { decay: 0.02 }, gain: 0.15 },
  },
});

const loaded = await loadPack("/packs/ui-sounds.json");
loaded.play("click");
```

## Sequences

Schedule sounds over time with `defineSequence`. Steps reference a sound definition, a time offset, and optional overrides. Loop the whole sequence or play it once.

```ts
import { defineSequence } from "audio-kit";

const play = defineSequence([
  { sound: { source: { type: "sine", frequency: 440 }, envelope: { decay: 0.1 }, gain: 0.3 }, time: 0 },
  { sound: { source: { type: "sine", frequency: 550 }, envelope: { decay: 0.1 }, gain: 0.3 }, time: 0.2 },
  { sound: { source: { type: "sine", frequency: 660 }, envelope: { decay: 0.1 }, gain: 0.3 }, time: 0.4 },
], { loop: true, bpm: 120 });

const stop = play();
```

## React hooks

`audio-kit/react` exports `useSound`, `usePack`, `useSequence`, and `useAnalyser`. Wrap your app with `SoundProvider` for global enabled/volume state.

```tsx
import { useSound } from "audio-kit/react";

function Button() {
  const play = useSound({
    source: { type: "sine", frequency: 880 },
    envelope: { decay: 0.1 },
    gain: 0.3,
  });

  return <button onClick={() => play()}>Click</button>;
}
```

## Offline rendering

Render a sound definition to an `AudioBuffer` or export it as a WAV file, without needing a user gesture or speakers.

```ts
import { renderToBuffer, renderToWav } from "audio-kit";

const buffer = await renderToBuffer({
  source: { type: "sine", frequency: 440 },
  envelope: { decay: 0.5 },
  gain: 0.3,
}, { duration: 1, sampleRate: 44100 });
```

## CLI

The `audio-kit` CLI manages sound packs from the command line.

```bash
npx audio-kit init        # scaffold a pack directory
npx audio-kit add click   # add a sound to the pack
npx audio-kit list        # list sounds in the pack
npx audio-kit remove click # remove a sound
```

---
title: loadPack
description: Load a patch from a URL or inline object.
---

```ts
import { loadPack } from "audio-kit";

const pack = await loadPack("/packs/minimal.json");
const voice = pack.play("tap");
```

**Parameters:**
- `source: string | SoundPack` - URL to a JSON file, or an inline patch object

**Returns:** `Promise<AudioPack>`

## Patch instance

| Method           | Returns                        | Description                |
| ---------------- | ------------------------------ | -------------------------- |
| `play(name)`     | `VoiceHandle`                  | Play a sound by name       |
| `get(name)`      | `SoundDefinition \| undefined` | Get the raw definition     |
| `sounds`         | `string[]`                     | List of sound names        |
| `ready`          | `boolean`                      | Whether the pack is loaded |
| `toJSON()`       | `SoundPack`                    | Serialize back to JSON     |
